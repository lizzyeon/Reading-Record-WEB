# BOOKSNAP

Instagram 클론코딩을 참고하여 만든 **독서 기록 웹 서비스**입니다.
사용자는 책 사진을 업로드하고 독서 기록을 남길 수 있습니다.
이 웹 서비스의 이름인 **BOOKSNAP** 또한 이러한 의미를 담고 있습니다.
다른 사용자의 기록을 피드 형식으로 확인할 수 있는 서비스를 목표로 합니다.

---

## Project Goal

- Django 기반 웹 서비스 구조 이해
- SNS 아키텍처 직접 구현
- 기능 단위 개발 및 Git 버전 관리 연습
- 실제 서비스 형태의 포트폴리오 구축

---
